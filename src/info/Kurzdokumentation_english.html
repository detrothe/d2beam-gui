Overview

This web application can be used to analyze 2D frames and trusses using the finite element method. Calculation options are available for static and dynamic analyses [1], [2], [3]. Static calculations can be performed according to first- and second-order theory. For dynamic calculations, eigenvalues and eigenmodes can currently be determined.
The beam element named d2beam (type 0) can take shear deformations into account according to Timoshenko's theory. The truss member is a classic 2-node element. The starting point for the d2beam element is a 4-node straight shear-flexible beam element [1], whose internal nodes are condensed so that the user works with a 2-node element.

After clicking the ‘Start new project’ button, you can choose between calculating a truss or a pure lattice structure. Selecting ‘Lattice structure’ simplifies the input for lattice structures. Of course, lattice structures can also be calculated with the truss elements if you enter moment joints for the members.
The elements have a start and end node, which are designated a and e, or 1 and 2.

Features of the program:

  • The cross-section is constant element by element.

  • Linearly elastic material for beams.

  • Shear deformations are taken into account using a shear correction factor. If a value of 0 is entered, the calculation is performed using shear-rigid elements.

  • In second-order theory, the imperfection can be conveniently taken into account via the first eigenform. For typical frame systems (fixed at the bottom, freely movable at the top) it is sufficient to specify the largest displacement without specifying the node (input 0). The direction of the displacement without specifying the node (input 0). The direction of the imperfection is set the same as the existing displacement of the corresponding degree of freedom from the previous calculation according to first-order theory, which is also required to determine the normal forces.
    For simple non-displaceable systems, the first eigenmode corresponds to a member pre-deformation and can be used accordingly.
    The pre-deformation calculated by the program can be checked in the graphical representation. If the result is not correct, the desired node with direction must be specified in the input.

  • Optional bedding of bending bars according to Winkler.

  • Load combinations with associated safety factors can be defined. In second-order theory, only the load combinations are calculated.



Graphical input

The structure is entered in the System tab. Depending on the available hardware, you can enter data using the mouse, the pen or your finger.

Please note the following points when entering data:

  • The cross-sections are defined in the Cross-sections tab. At least one cross-section must be created before entering the members. Changes can be made later in this tab.
  • When performing second-order calculations, only combinations of load cases that are defined in the Combinations tab prior to the calculation are calculated. The combinations must be defined in the Combinations tab prior to the calculation.
  • The graphical input for the system is located in the System tab.
  • The actual calculation is started in the Main tab. After successful calculation, the deformations, internal forces, etc. can be displayed in the Results tab.

Explanations of the icons in the System tab

Undo Button: Löscht das zuletzt erstellte Element (Knoten, Stab, Lager, Knotenlast oder Elementlast) und speichert es im internen Papierkorb.

Redo Button: Holt das zuletzt in den Papierkorb gestellte Element zurück.

Löschen Button: Löscht ein anzuklickendes Element und speichert es im internen Papierkorb.

Select Button: Es ist ein vorhandenes Element anzuklicken. Danach erscheint ein Dialog, in dem Änderungen vorgenommen werdn können.

Knoten Button: Eingabe von Knotenkoordinaten (x,z) mithilfe eines Dialogs. Hinweis: Die Eingabe von Knoten ist nur erforderlich, wenn ein Stabknoten nicht in die Schnittpunkte der Rasterlinien fällt.

Edit Knoten Button: Die Koordinaten (x,z) von vorhandenen Knoten können hier geändert werden.

Lager Button: Eingabe von Lagern. Hinweis: Lager können nur an vorhandene Knoten gesetzt werden.

Einstellungen Button: Hier können Einstellungen vorgenommen werden, die für die grafische Eingabe nützlich sind.

Reset Button: Das System wird neu gezeichnet. Zoom und Drag werden zurückgesetzt.

Default_Querschnitt Combobox: Der angezeigte Querschnittsname wird bei der nächsten Stabeingabe verwendet. Es können im Tab Querschnitte beliebig viele Querschnitte definiert werden.

Stab Button: Ein Stab wird durch Anfangs- und Endknoten definiert. Es gelten folgende Fangregeln: Zuerst wird innerhalb der Fangweite (Die Längen der Cursorlinien zeigen die Fangweite an) nach bereits vorhandenen Knoten gesucht und der am nächsten zum Cursorkreuz liegende Knoten verwendet. Wird nichts gefunden, dann werden die blauen Rasterpunkte verwendet. Wird auch hier nichts innerhalb der Fangweite gefunden, dann gilt die aktuelle Stelle des Cursors, gerundet auf 1 mm.

Knotenlast Button: Eingabe von Knotenlasten. Hinweis: Knotenlasten können nur an vorhandene Knoten gesetzt werden.

Elementlast Button: Eingabe von Stablasten. Hinweis: Stablasten können nur auf vorhandene Stäbe wirken.

Knotenmasse Button: Eingabe von Knotenmassen und Massenträgheitsmomenten. Hinweis: Knotenmassen können nur an vorhandene Knoten gesetzt werden. Sie werden nur berücksichtigt, wenn die Berechnungsart dynamisch im Tab Haupt gewählt wurde.

Weitere Hinweise zur Bedienung der App

Input

The comma or period is permitted as the decimal separator. A thousand separator is not permitted.

To speed up data entry, individual cell values or rows in the tables can be copied or generated. A block can be selected by pressing the RIGHT mouse button. After releasing the mouse button, a context menu appears. The context menu contains the following options:
For touch screens: When you swipe with your finger or stylus, the selected cells are highlighted in color, just like with a mouse. After lifting your finger/stylus, the context menu appears. During selection mode, no values can be entered. No values can be entered during selection mode. The close selection cancels the dialog and removes the selection.

  • copy first cell value: The value of the first selected cell is copied to the other selected cells.
  • copy first row values: At least two rows are to be selected. The values of the uppermost row are copied downwards column by column.
  • increment by 1: At least two rows are to be selected. The values of the top row are incremented by 1 column by column
  • increment by delta: At least three rows are to be selected. The differences between the top row and the row below are determined column by column. The rows below are changed column by column with the calculated delta values.
  • copy to clipboard: The content of the selected cells is copied to the clipboard and can be used in other places in the table, for example. The values can also be transferred to other programs (spreadsheet).
  • insert from clipboard: The clipboard content is pasted starting at the upper left cell of the selected area.
  • close: The dialog is ended without further explanations.

For iOS, the selection in the context menu is made with a double click.

Note: A Bluetooth mouse and/or keyboard can be connected to any tablet or smartphone.

To make better use of the limited screen space on smartphones and tablets, the number of rows in the tables for node coordinates and elements are determined by entering the number of nodes and elements. Therefore, the new number of elements and nodes must be entered first and then the size of the tables must be changed using the 'resize tables' button. The process can be repeated as often as necessary.

2D representations

The display options can be combined at will.

  • labeling: Labeling of elements, internal forces and deformations. The type of labeling depends on the selection of additional options.
  • systemlines: Show structural system.
  • deformations: Show deformed structure.
  • normal force: axial force diagram
  • shear force: shear force diagram
  • moment: moment diagram
  • Eigenform: Display of eigenforms. In the selection box at the top right, you can select from the calculated eigenforms.
  • imperfection: Representation of the imperfection used, based on the first eigenform. Please note that the imperfection used depends on the combination selected. Only for second-order theory.
  • local imperfection: Display of the entered local imperfections, only for theory II. Order.
  • scaling: Scaling of the size of internal force diagrams and deformations.
  • show loads: Display of loads depending on the selected load case.
  • show supports: Display of calculated support forces and symbols.
  • show outline: The cross-section heights of the bars entered are used in the display.
  • total deformation: Only for Th. II. Order. Normally, the relative deformations are displayed. If the option is selected, the pre-deformations are added to the relative deformations.
  • equilibrium of forces: Only for Th. 2nd Order. If this option is selected, the internal forces along the member are displayed in the directions of the local coordinate system of the undeformed system (equilibrium internal forces). If the option is not selected, the equilibrium internal forces N and Vz are transformed in the direction of the rotated cross-section. These are also referred to as design internal forces and are to be used for the design.
    This option has no effect on the bending moment My !
  • Reset: Settings will be reset.

Save graphic

With the save svg button, you can save the current graphic representation in a file in svg format (vector format). The drawing can be opened with any program that supports the svg vector format , such as Inkscape, Libre Office, and Microsoft Office (from version 2016 onwards). Since it is a vector format, the graphic can be edited.

Installing the app on your desktop

This application is programmed as a web application (web app), so that it can be installed on the desktop under all operating systems. An icon is created on the desktop for this purpose. This allows the An icon is created on the desktop for this purpose. This means that the app behaves like an app installed from a store . D2beam-gui will of course continue to run in the browser engine of the installed browser, but without controls such as the address bar. The installation differs depending on the operating system, version, and browser.
After d2beam has been started in the browser for the first time, select the ‘Install app’ option from the application menu (three-dot menu) in the top right-hand corner of the Chrome browser on Android. the option ‘Install app’. A dialog box will appear, which must be confirmed. The browser can then be closed. A new icon named D2beam-gui will now appear on the desktop. The icon can be used to start the application.

For installation on iOS, you must use the Safari browser. Here, the command is called “Add to Home Screen” with a plus sign icon.

Installation on Microsoft Windows is similar to installation on Android. In the Edge browser, the following section is available in the application menu (three-dot menu)
In newer versions of Chrome, you may be asked directly whether you want to install it. See the icon on the right in the address bar:

On macOS, the Microsoft Edge or Google Chrome browser should be used for installation.

The Firefox browser does not currently support installation.
On Microsoft Windows and Linux PC operating systems, you can alternatively switch to full-screen mode by pressing the F11 key.

On Android, iOS, and Linux, uninstallation is done by simply deleting the icon on the desktop. On Windows, the app is uninstalled like a normal program.

Virtual keyboard

On some Android devices, especially from Samsung, the numeric keyboard does not include a key for a negative sign. Unfortunately, the HTML specification leaves it up to the device manufacturer whether the numeric keyboard contains a key for the negative sign. Judging from the many Internet entries, the deeper meaning behind this is probably only clear to a few people. In these cases, it is recommended to install an additional virtual keyboard. For example, the original Google Android keyboard Gboard from the Play Store is an alternative. The keyboard still has to be activated after the installation. It is best to do this right away in the Play Store. Later, you can always switch between the installed keyboards (icon at the bottom right of the keyboard).

Questions and answers

  • Question: I have entered a few too many nodes that I no longer need.
    Answer: That doesn't matter. Nodes that don't have any elements 'attached' to them are ignored.
  • Question: What is the node ID?
    Answer: Each node has its own internal number (node ID), which is automatically generated when the node is created. It is independent of the subsequent assignment of the FE node numbering. The node ID is displayed in the node dialog in the System tab when you select the Edit Node button and click on an existing node.

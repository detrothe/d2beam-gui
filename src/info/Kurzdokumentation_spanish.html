Resumen

Esta aplicación web se puede utilizar para analizar estructuras y vigas en 2D utilizando el método de elementos finitos. Hay opciones de cálculo disponibles para análisis estáticos y dinámicos [1], [2], [3]. Los cálculos estáticos se pueden realizar según la teoría de primer y segundo orden. Para los cálculos dinámicos, actualmente se pueden determinar los valores propios y los modos propios. El elemento de viga denominado d2beam (tipo 0) puede tener en cuenta las deformaciones por cizallamiento según la teoría de Timoshenko. El miembro de la armadura es un elemento clásico de 2 nodos. El punto de partida para el elemento d2beam es un elemento de viga recta flexible al corte de 4 nudos [1], cuyos nudos internos se condensan para que el usuario trabaje con un elemento de 2 nudos. Para los cálculos según la teoría de primer orden, las matrices de elementos corresponden a la solución obtenida utilizando la ecuación diferencial para la viga de Timoshenko. En este caso, los resultados son, por lo tanto, exactos.

Después de hacer clic en el botón «Iniciar nuevo proyecto», se puede elegir entre el cálculo de una estructura de barras o una estructura de celosía pura. La opción «Estructura de celosía» simplifica la introducción de datos en estructuras de celosía. Por supuesto, con los elementos de estructura de barras también se pueden calcular estructuras de celosía si se introducen articulaciones de momento para las barras.
Los elementos tienen un nodo inicial y un nodo final, que se designan con a y e, o 1 y 2.

Características del programa:

  • La sección transversal es constante en todos los elementos.

  • Material linealmente elástico para barras.

  • Las deformaciones por empuje se tienen en cuenta mediante un factor de corrección de empuje. Si se introduce un valor de 0, se calcula con elementos rígidos al empuje.

  • En la teoría de segundo orden, la imperfección se puede tener en cuenta fácilmente a través de la primera forma propia. En los sistemas de bastidores típicos (fijados en la parte inferior y móviles en la parte superior), basta con indicar el mayor desplazamiento sin especificar el nodo (entrada 0). La dirección de la imperfección se establece igual que el desplazamiento existente del grado de libertad correspondiente del cálculo anterior según la teoría de primer orden, que también se necesita para determinar las fuerzas normales.
    En sistemas simples no desplazables, la primera forma propia corresponde a una deformación previa de la barra y puede utilizarse en consecuencia.
    La deformación previa calculada por el programa se puede verificar en la representación gráfica . Si el resultado no es correcto, se debe especificar el nodo deseado con la dirección en la entrada.

  • Apoyo opcional de las barras de flexión según Winkler.

  • Se pueden definir combinaciones de carga con los factores de seguridad correspondientes. En el caso de la teoría de segundo orden, solo se calculan las combinaciones de carga.



Entrada gráfica

La introducción del sistema estático se realiza en la pestaña Sistema. Dependiendo del hardware disponible, se puede introducir con el ratón, el lápiz o el dedo.

Folgenden Punkte sind bei der Eingabe zu beachten:

  • Die Querschnitte werden im Tab Querschnitte definiert. Mindestens ein Querschnitt muss vor der Eingabe der Stäbe erstellt werden. Änderungen sind nachträglich in diesem Tab möglich.
  • Bei Berechnungen nach Theorie II. Ordung werden nur Kombinationen der Lastfälle berechnet, die vor der Berechnung im Tab Kombinationen definiert werden müssen.
  • Die grafische Eingabe des Systems findet im Tab System statt.
  • Die eigentliche Berechnung wird im Tab Haupt gestartet. Nach erfolgreicher Berechnung können im Tab Ergebnisse die Verformungen, Schnittgrößen etc. dargestellt werden.

Erläuterungen zu den Icons im Tab System

Undo Button: Löscht das zuletzt erstellte Element (Knoten, Stab, Lager, Knotenlast oder Elementlast) und speichert es im internen Papierkorb.

Redo Button: Holt das zuletzt in den Papierkorb gestellte Element zurück.

Löschen Button: Löscht ein anzuklickendes Element und speichert es im internen Papierkorb.

Select Button: Es ist ein vorhandenes Element anzuklicken. Danach erscheint ein Dialog, in dem Änderungen vorgenommen werdn können.

Knoten Button: Eingabe von Knotenkoordinaten (x,z) mithilfe eines Dialogs. Hinweis: Die Eingabe von Knoten ist nur erforderlich, wenn ein Stabknoten nicht in die Schnittpunkte der Rasterlinien fällt.

Edit Knoten Button: Die Koordinaten (x,z) von vorhandenen Knoten können hier geändert werden.

Lager Button: Eingabe von Lagern. Hinweis: Lager können nur an vorhandene Knoten gesetzt werden.

Einstellungen Button: Hier können Einstellungen vorgenommen werden, die für die grafische Eingabe nützlich sind.

Reset Button: Das System wird neu gezeichnet. Zoom und Drag werden zurückgesetzt.

Default_Querschnitt Combobox: Der angezeigte Querschnittsname wird bei der nächsten Stabeingabe verwendet. Es können im Tab Querschnitte beliebig viele Querschnitte definiert werden.

Stab Button: Ein Stab wird durch Anfangs- und Endknoten definiert. Es gelten folgende Fangregeln: Zuerst wird innerhalb der Fangweite (Die Längen der Cursorlinien zeigen die Fangweite an) nach bereits vorhandenen Knoten gesucht und der am nächsten zum Cursorkreuz liegende Knoten verwendet. Wird nichts gefunden, dann werden die blauen Rasterpunkte verwendet. Wird auch hier nichts innerhalb der Fangweite gefunden, dann gilt die aktuelle Stelle des Cursors, gerundet auf 1 mm.

Knotenlast Button: Eingabe von Knotenlasten. Hinweis: Knotenlasten können nur an vorhandene Knoten gesetzt werden.

Elementlast Button: Eingabe von Stablasten. Hinweis: Stablasten können nur auf vorhandene Stäbe wirken.

Knotenmasse Button: Eingabe von Knotenmassen und Massenträgheitsmomenten. Hinweis: Knotenmassen können nur an vorhandene Knoten gesetzt werden. Sie werden nur berücksichtigt, wenn die Berechnungsart dynamisch im Tab Haupt gewählt wurde.

Weitere Hinweise zur Bedienung der App

Eingabe

Als Dezimal-Trennzeichen ist das Komma oder der Punkt zulässig. Ein Tausender-Trennzeichen ist nicht zulässig.

Um die Eingabe zu beschleunigen, können einzelne Zellwerte oder Zeilen der Tabellen kopiert bzw. generiert werden. Es kann durch Drücken der RECHTEN Maustaste ein Block selektiert werden. Nach dem Loslassen der Maustaste erscheint ein Kontextmenü.
Bei Touch screens: Beim Übersteichen mit dem Finger oder dem Stift werden dann wie bei der Maus die selektierten Zellen farblich markiert. Nach Heben des Fingers/Stifts erscheint das Kontextmenu. Während des Selektiermodus können keine Werte eingegeben werden. Die Auswahl close bricht den Dialog ab und entfernt die Selektion.

  • copy first cell value: Es wird der Wert der zuerst selektierten Zelle in die anderen selektierten Zellen kopiert.
  • copy first row values: Es sind mindestens zwei Zeilen zu selektieren. Es werden die Werte der obersten Zeile spaltenweise nach unten kopiert.
  • increment by 1: Es sind mindestens zwei Zeilen zu selektieren. Es werden die Werte der obersten Zeile spaltenweise um 1 erhöht.
  • increment by delta: Es sind mindestens drei Zeilen zu selektieren. Es werden spaltenweise die Differenzen zwischen der obersten und der darunterliegenden Zeile bestimmt. Die darunterliegenden Zeilen werden spaltenweise mit den berechneten Deltawerten verändert.
  • copy to clipboard: Der Inhalt der selektierten Zellen wird in die Zwischenablage kopiert und kann zum Beispiel an anderen Stellen der Tabelle eingesetzt werden. Auch können die Werte an andere Programme übergeben werden (Tabellenkalkulation)
  • insert from clipboard: Der Inhalt der Zwischenablage wird beginnend an der linken oberen Zelle des markierten Bereichs eingefügt.
  • close: Der Dialog wird ohne weitere Ausführungen beendet.

Bei iOS erfolgt die Auswahl im Kontextmenü mit einem Doppelklick.

Hinweis: An jedes Tablett und Smartphone läßt sich eine Bluetooth Maus und/oder Tastatur anschließen.

Um den begrenzten Bildschirmplatz auf Smartphones und Tabletts besser zu nutzen, werden die Anzahl der Zeilen in den Tabellen für die Knotenkoordinaten und Elemente durch die Eingabe der Anzahl Knoten und Elemente bestimmt. Deshalb sind erst die neue Anzahl Elemente und Knoten einzugeben und dann die Größe der Tabellen mit Hilfe des Buttons 'Tabelle anpassen' zu änden. Der Vorgang kann beliebig oft wiederholt werden.

Darstellungen 2D

Die Darstellungsoptionen können beliebig kombiniert werden.

  • Beschriftung: Beschriftung der Elemente. Schnittgrößen und Verformungen. Die Art der Beschriftung ist abhängig von der Wahl weiterer Optionen.
  • Systemlinien: Darstellung des Systems.
  • Verformungen: Darstellung des verformten Systems.
  • Normalkraft: Zustandskinien für die Normalkräfte
  • Querkraft: Zustandslinien für die Querkräfte
  • Moment: Zustandslinien für die Momente
  • Eigenformen: Darstellung der Eigenformen. In der Auswahlbox rechts oben können Sie aus den berechneten Eigenform auswählen.
  • Schiefstellung: Darstellung der verwendeten Schiefstellung, basierend auf der ersten Eigenform. Achtung, die verwendete Schiefstellung ist abhängig von der eingestellten Kombination. Nur bei Theorie II. Ordnung.
  • Stabvorverformung: Darstellung der eingegebenen Stabvorverformungen, nur bei Theorie II. Ordnung.
  • Skalierung: Skalierung der Größe der Schnittgrößen und Verformungen.
  • Lasten anzeigen: Darstellung der Lasten in Abhängigkeit von dem gewählten Lastfall.
  • Lagerkräfte anzeigen: Darstellung der berechneten Lagerkräfte und Symbole.
  • Umriss anzeigen: Die eingegebenen Querschnittshöhen der Stäbe werden bei der Darstellung verwendet.
  • Gesamtverformung: Nur bei Th. II. Ordnung. Normalerweise werden die relativen Verformungen angezeigt. Wenn die Option gewählt wird, dann werden die Vorverformungen zu den relativen Verformungen addiert.
  • Gleichgewicht SG: Nur bei Th. II. Ordnung. Wenn diese Option gewählt wird, dann werden die Schnittgrößen entlang des Stabes in den Richtungen des lokalen Koordinatensystens des unverformten Systems angezeigt (Gleichgewichtsschnittgrößen). Wenn die Option nicht ausgewählt ist, dann werden die Gleichgewichtsschnittgrößen N und Vz in die Richtung des gedrehten Querschnitts transformiert. Diese werden auch als Nachweisschnittgrößen bezeichnet und sind für die Bemessung zu verwenden.
    Diese Option hat keine Auswirkung auf das Biegemoment My!
  • Reset: Einstellungen werden zurückgesetzt.

Grafik speichern

Mit dem Button save svg kann man die aktuelle grafische Darstellung in einer Datei im svg-Format (Vektorformat) speichern. Die Zeichnung kann mit jedem Programm, das das Vektorformat svg unterstützt, zum Beispiel Inkscape, Libre Office und Microsoft Office (ab Version 2016) als Bild eingelesen werden. Da es sich um ein Vektorformat handelt, kann die Grafik bearbeitet werden.

Voreinstellungen

In dem ꔷꔷꔷ Punkte Menü kann die Schriftgröße und die Tabellenfarbe eingestellt werden.
Die Einstellungen der Seite können als Voreinstellung im lokalen Speicher des verwendeten Browsers gespeichert werden und stehen nach einem Programmneustart wieder zur Verfügung. Die Voreinstellungen können mit dem darunterliegenden Button wieder gelöscht werden.
Bei dem lokalen Speicher (local storage) handelt es sich nicht um Cookies. Diese Webanwendung führt alle Berechnungen auf dem lokalen Rechner im Browser in JavaScript durch. Es werden keine Daten mit einem Webserver ausgetauscht. Nach dem Programmstart ist deshalb keine Netzwerkverbindung mehr erforderlich.

Installation als App auf dem Desktop

Diese Anwendung ist als Webanwendung (Web App) programmiert, so dass sie unter allen Betriebssystemen auf dem Desktop installiert werden kann. Dafür wird auf dem Desktop ein Icon erstellt. Damit verhält sich die App wie eine aus einem Store installierte App. D2beam wird natürlich weiterhin im installierten Browser ausgeführt, allerdings ohne Bedienelemente wie zum Beispiel die Adressleiste. Die Installation unterscheidet sich je nach Betriebssystem, -version und Browser.
Nachdem d2beam erstmalig im Browser gestartet wurde, wählt man unter Android im Chrome Browser rechts oben aus dem Anwendungsmenü (Drei-Punkte-Menü) die Option 'App installieren'. Ein Dialog erscheint, der zu bestätigen ist. Danach kann der Browser geschlossen werden. Auf dem Desktop gibt es jetzt ein neues Icon mit Namen d2beam.

Für die Installation unter iOS ist der Safari Browser zu verwenden. Hier heißt der Befehl 'Zum Home-Bildschirm' mit einem Addieren-Symbol

Die Installation unter Microsoft Windows ist ähnlich der Installation unter Android. Im Edge Browser steht folgender Abschnitt im Anwendungsmenü (Drei-Punkte-Menü)
Bei neueren Chrome Versionen werden Sie unter Umständen direkt gefragt, ob eine Installation gewünscht wird, siehe Symbol rechts in der Adressleiste :

Unter macOS ist der Microsoft Edge or Google Chrome Browser zur Installation zu verwenden.

Der Firefox Browser unterstützt die Installation zur Zeit noch nicht.
Unter den PC-Betriebsystemen Microsoft Windows und Linux kann alternativ mit der F11-Taste in den Vollbildmodus umgeschaltet werden.

Eine Deinstallation erfolgt durch einfaches Löschen des Icons auf dem Desktop.

Teclado virtual

En algunos dispositivos Android, especialmente los de Samsung, el teclado numérico no incluye una tecla para el signo negativo . Desafortunadamente, la especificación HTML deja a criterio del fabricante del dispositivo si el teclado numérico contiene una tecla para el signo negativo. A juzgar por las numerosas entradas en Internet , es probable que solo unas pocas personas conozcan el significado más profundo que se esconde detrás de esto. En estos casos, se recomienda instalar un teclado virtual adicional . Por ejemplo, el teclado original de Google Android, Gboard, de Play Store es una alternativa. El teclado aún debe activarse después de la instalación. Es mejor hacerlo de inmediato en Play Store. Más adelante, siempre puede cambiar entre los teclados instalados (icono en la parte inferior derecha del teclado).

Preguntas y respuestas

  • Pregunta: He introducido algunos nodos de más que ahoraya no necesito.
    Respuesta: No pasa nada. Los nodos que no tienen elementos «colgados» se ignoran.
  • Pregunta: ¿Qué es el ID de nodo?
    Respuesta: Cada nodo tiene un número interno propio (ID de nodo) que se genera automáticamente al crear el nodo. Es independiente de la posterior asignación de la numeración de nodos FE. El ID de nodo se muestra en la pestaña Sistema del cuadro de diálogo Nodo cuando seleccionas el botón Editar nodo y haces clic en un nodo existente.
